# Apple's Liquid Glass Inspired - Interactive

A Pen created on CodePen.

Original URL: [https://codepen.io/prxju/pen/vEOjMvr](https://codepen.io/prxju/pen/vEOjMvr).

A sleek, interactive glass effect inspired by Apple's Liquid Glass design. This component bends light realistically, distorting the background without frosting. Fully movable, touch-friendly, and powered by SVG filters + CSS transparency for a futuristic UI look!
