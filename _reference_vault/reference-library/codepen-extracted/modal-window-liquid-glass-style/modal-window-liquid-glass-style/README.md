# Modal Window Liquid Glass style

A Pen created on CodePen.

Original URL: [https://codepen.io/sanyi-dalmadi/pen/WbrzRmz](https://codepen.io/sanyi-dalmadi/pen/WbrzRmz).

A two-layered liquid glass modal window with parallax motion, blurred backdrop, and noise-masked frosted effect. The modal reacts to mouse movement with layered depth and edge offset, simulating physical glass thickness. Includes a matching open button with masked SVG noise and teachable visual logic.

Inspired by frosted UI surfaces and dungeon-themed overlays.

Noise filter via inline SVG (feTurbulence), layered motion via CSS transform, and accessibility-ready modal structure.

Credits: SVG noise technique adapted from community examples.
